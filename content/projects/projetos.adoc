== Meus Projetos

* **link:https://arthurfelipercosta.github.io/projects/financeiro.html[Financeiro AI Dashboard]** +
Plataforma completa de gestão financeira familiar desenvolvida com **React** e **TypeScript**. Integra o **Google Gemini API** para fornecer insights inteligentes sobre gastos e **Firebase (Realtime Database & Auth)** para sincronização multiusuário em tempo real. Possui lógica avançada para gestão de parcelamentos, cartões de crédito e visualização de dados dinâmica com **Recharts**.

* **link:https://github.com/arthurfelipercosta/minhas-leituras[Minhas Leituras]** +
Ecossistema mobile completo para gestão literária desenvolvido com **React Native**, **Expo (SDK 54)** e **TypeScript**. Implementa uma arquitetura robusta baseada em **Context API** para gestão de autenticação e temas dinâmicos (**Dark/Light Mode**). Utiliza **Firebase** para sincronização em nuvem e **AsyncStorage** para persistência offline, contando com módulos de estatísticas avançadas, integração com APIs de livros e sistema de notificações via Toast.

* **link:https://github.com/arthurfelipercosta/hotwheelsapi[Hot Wheels API]** +
Infraestrutura de dados Open-Source projetada para catalogar o ecossistema Hot Wheels com alta granularidade. Implementa uma arquitetura de API Estática (Data-as-Code) fundamentada em JSON, utilizando lógica de normalização para desassociar moldes (castings) de variações específicas (releases). O projeto conta com suporte nativo a internacionalização (i18n), validação rigorosa via JSON Schema para contribuições da comunidade e automação em Python para web scraping e processamento de dados em larga escala. +
(An Open-Source data infrastructure designed to catalog the Hot Wheels universe with high granularity. Built on a Static API (Data-as-Code) architecture, it leverages JSON to implement data normalization, separating historical castings from seasonal releases. The project features native internationalization (i18n), JSON Schema validation to ensure the integrity of community-driven contributions, and Python-based automation for large-scale data extraction and processing.)
